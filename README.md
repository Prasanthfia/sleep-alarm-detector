# 😴 Sleep Alarm Detector

A real-time sleep detection system that monitors eye movements through a webcam and alerts users when prolonged eye closure is detected. The application uses computer vision techniques to track facial landmarks and determine whether the user is awake or falling asleep.

---

## 🎯 Purpose of the Project

The purpose of this project is to help users stay alert while studying, working, attending online classes, or performing tasks that require continuous attention.

The system detects prolonged eye closure and immediately triggers an alarm to help prevent drowsiness and improve focus.

---

## 🚀 Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy
- Pygame
